== Getting Started ==
Memberlite [Coffee] makes it easy to customize the appearance and layout of your site using the Theme Customization Screen (Appearance > Customize). 

== Adding Your Logo ==
Use the Customize > Site Identity screen to add a custom logo (formatted for retina display), update your Site Title and Tagline, and toggle the display of your Site Title & Tagline. Your admin email's gravatar will be used by default.
